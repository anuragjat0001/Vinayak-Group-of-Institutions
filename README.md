🎓 Vinayak Group of Education — Official Website

A premium, multi-page educational website built for Vinayak Group of Education, Chomu (Jaipur, Rajasthan).

---

🚀 Project Overview

This website represents a complete digital presence for an educational group comprising:

- CBSE Schools
- RBSE School
- UG & PG Colleges
- Girls College
- B.Ed Teacher Training Institute
